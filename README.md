
## Description

This project updates user wallet ballance using prisma transaction.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run prisma migration

```bash
# development
$ yarn migrate:dev

# watch mode
$ yarn start:dev

# production mode
$ yarn migrate:deploy
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
